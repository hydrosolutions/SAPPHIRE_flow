# Evidence Memos

This folder sits **alongside** the existing thematic review files
([01-operational-systems.md](../source-reviews/01-operational-systems.md)
through
[06-datasets-and-nwp.md](../source-reviews/06-datasets-and-nwp.md)). The older
files remain
the broad research notebooks; these new memos are the evidence-first layer that
will feed the scoping review and later manuscript draft.

## Purpose

Each memo should do four things:

1. distil the strongest supported findings from the corresponding legacy review
2. record a focused research top-up based on additional primary sources
3. separate established findings from limited evidence and open questions
4. identify the next study-level extraction targets and verification risks

## Status labels used in the memos

- `Established finding`: supported by multiple verified sources
- `Promising but limited evidence`: supported, but only indirectly or in a few
  constrained studies
- `Open question`: still unresolved after the current search
- `Provisional claim`: plausible, but still dependent on unresolved checks or
  preprints

## How to use these memos

- Keep the older `01` to `06` files as source notebooks.
- Use the memos for scoping-review drafting and claim selection.
- Promote individual papers from a memo into study-level records using
  [evidence-extraction-template.md](../evidence-extraction-template.md).
- Do not move a claim into publication-ready prose until the memo no longer
  treats it as provisional.
